QupZilla-Qt - lightweight web browser. Modified version of QupZilla browser.  

Plugins:
 - AdBlock
 - Greasemonkey / Userscripts ([UserScripts.md](userscripts/))
 - Toggle JavaScript
 - Network status
 - User Agent Manager
 - Vim Plugin
 - Vertical Tabs
 - Tab Manager
